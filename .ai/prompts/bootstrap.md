# Bootstrap Prompt

Establish repository context before starting a new work session.

1. Read `.ai/context/PROJECT_CONTEXT.md`, `.ai/context/ACTIVE_TASK.md`, and `.ai/memory/KNOWN_ISSUES.md`.
2. Read the relevant documents listed in `docs/00_DOCUMENT_INDEX.md`.
3. Confirm the authorized objective, scope, constraints, and approval boundaries.
4. Identify affected files, dependencies, risks, and validation steps.
5. Do not implement changes until the task is understood and authorized.
