# Tests

## Purpose

Provide the home for automated verification of the Enterprise PSEO framework.

## Responsibilities

Contain unit, integration, contract, end-to-end, fixture, and test-support artifacts that demonstrate expected behavior.

## What Belongs Here

Tests, non-sensitive fixtures, mocks, test utilities, and reproducible test data required by approved validation suites.

## What Must Never Be Placed Here

Production secrets, live customer data, application source modules, manually maintained production artifacts, or tests that depend on undocumented external state.
