# Source

## Purpose

Provide the presentation-layer home for the Enterprise PSEO application.

## Responsibilities

Organize layouts, route pages, reusable components, presentation assets, and presentation-specific data adapters.

## What Belongs Here

- `layouts/` — shared page shells and composition boundaries.
- `pages/` — route-level presentation modules.
- `components/` — reusable user-interface components.
- `assets/` — source presentation assets.
- `data/` — presentation-facing data mappings; not raw source datasets.

## What Must Never Be Placed Here

Engine business logic, raw datasets, secrets, deployment artifacts, generated production output, or unrelated scripts.
